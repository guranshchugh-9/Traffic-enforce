## CA-SORT - Cross Association SORT

This repository provides the implementation of **CA-SORT (Cross Association-SORT)**, as described in the paper *"DashCop: Automated E-ticket Generation for Two-Wheeler Traffic Violations Using Dashcam Videos"*.

### Overview

The primary objective of CA-SORT is to jointly track two data classes that exhibit a correspondence relationship, such as:

- **Rider and Bike**
- **Person and Skateboard**
- **Person and Chair**

Unlike standard SORT (Simple Online and Realtime Tracking) algorithms and similar trackers that handle multiple objects independently, CA-SORT focuses on tracking pairs of associated objects together, leveraging their cross-class relationships to enhance tracking accuracy and consistency.

### Key Features

- Joint tracking of two associated object classes.
- Integration of segmentation masks and cross-masks for improved association.

## Usage

### Main File
The main entry point for the tracker is **`assoc_tracker.py`**, which contains the implementation of the tracking algorithm.

### Initialization
To use the tracker, initialize an instance of the `AssocTracker` class:

```python
tracker = AssocTracker(cfg)
```

Here, `cfg` contains the hyperparameters for the tracker. Refer to the provided `base_cfg.yaml` or `base_cfg.json` files for configuration details.

### Tracking Detections
To track detections in a given frame, use the `track` method:

```python
tracker_assoc.track(all_preds_data, all_masks, all_cross_masks, frame)
```

#### Input Parameters:
- **`all_preds_data`**: A NumPy array of shape `(N, 6)` containing detection data for all objects:
  - Bounding box coordinates in **(xyxy)** format.
  - Confidence score for the detection.
  - Class ID of the detection (e.g., `0` for rider, `1` for motorbike).

- **`all_masks`**: A NumPy array of shape `(N, H, W)` containing segmentation masks for the `N` detected objects.

- **`all_cross_masks`**: A NumPy array of shape `(N, H, W)` containing cross-masks (e.g., masks generated by SAC).

- **`frame`**: The current image frame as a NumPy array of shape `(H, W, 3)`.

#### Function Behavior:
The `track` method updates the state of the tracker by:
- Matching existing tracks with new detections.
- Creating new tracks when necessary.

### Retrieving Tracks
To obtain the current tracks at any point, use the `give_tracks` method:

```python
tracks_rider, tracks_motor, motor_rider_count = tracker_assoc.give_tracks()
```

#### Output:
- **`tracks_rider`**: Current tracks for the rider class.
- **`tracks_motor`**: Current tracks for the motorbike class.
- **`motor_rider_count`**: The count of associated riders with each motorbike.
